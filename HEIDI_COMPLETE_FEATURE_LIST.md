## Product Overview

**Heidi Health** is an AI-powered medical scribe platform that transforms clinical documentation by automating the creation of clinical notes and letters during patient encounters. Designed for clinicians across 200+ medical specialties in 100+ countries.

### Key Statistics
| Metric | Value |
|--------|-------|
| Consults supported weekly | 2 million+ |
| Hours returned to clinicians yearly | 8 million+ |
| Languages supported | 110+ |
| Medical specialties | 200+ |
| Countries | 100+ |
| Documentation time reduction | 61% |
| Patient acceptance rate | 100% |

## Core Products

### 1. AI Clinical Scribe
The primary product that listens to patient consultations and automatically generates clinical documentation.

### 2. AI Coding Assistant
Automatically suggests billing codes (ICD-10, SNOMED-CT) based on consultation content.

### 3. Document Generator
Creates referral letters, patient summaries, care plans, certificates, and more from consultation data.

### 4. Template Library
Thousands of customizable templates for all specialties and document types.

### 5. EHR Integration Suite
Direct integrations with major EHR systems for seamless workflow.

## Transcription & Documentation

### Real-Time Transcription
- **Languages:** 110+ supported
- **Speaker Diarization:** Automatically identifies Doctor vs Patient speech
- **Accuracy:** Handles background noise and language accents
- **Offline Mode:** Continues working without internet, syncs when reconnected

### Audio Capture
| Feature | Description |
|---------|-------------|
| Start/Stop Controls | One-click recording toggle |
| Visual Indicator | 3-bar audio waveform showing capture status |
| Timer Display | Real-time duration counter |
| Pause Function | Temporarily pause without ending session |
| Device Selection | Choose specific microphone input |

### Context Section (Silent Notes)
Add information not spoken aloud during the consultation:
- Physical observations
- Vital signs
- Non-verbal cues
- Reference information

### File Uploads
Supported formats for additional context:
- PDFs
- DOC/DOCX files
- Images (JPG, PNG, etc.)
- Handwritten notes (scanned)
- Direct photo capture

## Note Generation

### Automatic Note Creation
1. Start transcribing consultation
2. Stop recording when complete
3. Select template (or use default)
4. Heidi generates structured note in seconds
5. Review, edit, and export

### Note Structure Options
- **SOAP** - Subjective, Objective, Assessment, Plan
- **SOAP (Issues)** - Includes Problem List
- **H&P** - History & Physical
- **Progress Note** - Brief follow-up documentation
- **Procedure Note** - Document procedures and interventions
- **DAP Note** - Data, Assessment, Plan
- **Session Summary** - Therapy/counseling sessions
- **Pre/Post-Op Reports** - Surgical documentation
- **Telehealth Visit** - Remote consultation
- **Discharge Summary** - Patient discharge documentation

### Regeneration
- Regenerate entire note with different template
- Regenerate specific sections
- Adjust detail level with Brain settings

## Ask Heidi AI Assistant

### Capabilities
Ask Heidi is a voice-supported AI command bar that can:
- Modify existing notes
- Add specific details
- Change formatting
- Split notes into sections
- Generate additional documents
- Fill out forms
- Answer clinical questions

### Example Commands
```
"Add more detail to the assessment section"
"Write a referral letter to cardiology"
"Make the plan more concise"
"Add medication dosages"
"Generate a patient handout"
"Fill out the Medicare form"
```

### Availability
- Included in Pro, Together, and Enterprise plans
- Accessible via web, desktop, mobile, and Chrome extension

## Billing & Coding

### Automatic Code Generation
After processing consultation, Heidi automatically generates suggested codes in the background.

### Supported Code Sets
| Code Set | Region/Use |
|----------|------------|
| ICD-10-CM | US clinical diagnoses |
| ICD-10-UK | UK healthcare |
| ICD-10-AM | Australian healthcare |
| SNOMED-CT | International clinical terminology |
| HCPCS | US Medicare/Medicaid |

*Note: CPT codes support expanding*

### Code Review Interface
| Feature | Description |
|---------|-------------|
| Relevance Ranking | High, Medium, Low relevance indicators |
| Billable Indicator | $ symbol for billable codes |
| Evidence Linking | Click to see supporting note text |
| Replace Function | Search and swap codes |
| Confirmation Status | Color-coded progress (Red/Orange/Green) |

### Code Management
1. View suggested codes after note generation
2. Review relevance and evidence
3. Confirm, replace, or remove codes
4. Export to billing system

## Magic Paste

### Overview
One-click transfer of notes to any EHR system with intelligent field mapping.

### How It Works
1. Copy note or click Magic Paste
2. Right-click in target EHR field
3. Select "Heidi Magic Paste" from context menu
4. Note content automatically inserted and formatted

### Features
- Intelligent field detection
- Format preservation
- Section mapping
- Works with any text field
- Backup context menu option

## Brain Settings (Scribe Modes)

### Standard Scribe (formerly Left Brain)
- **Characteristics:** Fast, precise, concise
- **Best for:** Most consultations, simple visits
- **Output:** Efficient, focused documentation
- **Availability:** All users

### Extended Scribe (formerly Right Brain)
- **Characteristics:** Thoughtful, detailed, comprehensive
- **Best for:** Complex cases, longer consultations
- **Output:** More thorough documentation
- **Availability:** Pro, Together, Enterprise plans

### Setting Your Default
Navigate to: Settings > Preferences > Defaults
